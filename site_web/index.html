    Bienvenue dans le monde de l'intelligence artificielle Pac-Man.

    Nous avons pour objectif de créer un IA qui apprendra à jouer au jeu Pac-man.

    Ce site sert de complément d'information sur notre projet et détaille certains éléments comme une petite présentation de chacun d'entre nous et nos rapports de soutenance.

    En première partie, l'objectif est de recréer le jeu Pac-man le plus fidèlement possible autant dans les mécaniques que dans les graphismes. Puis, nous nous consacrerons au développement de l'IA. Nous allons créer un réseau de neurones utilisant le Q-learning pour apprendre. Nous ferons donc du deep semi-supervized machine learning.
