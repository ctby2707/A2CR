    Bienvenue dans le monde de l'intelligence artificielle Pac-Man.

    Nous avons pour objectif de creer un IA qui apprendra a jouer a Pac-man.

    Ce site sert de complement d'information sur notre projet et detaille certain element comme une petite presentation de chacun d'entre nous et nos rapports de soutenance.

    En premiere partie, l'objectif est de recreer le jeu pac-man le plus fidelement autant dans les mecaniques que dans les graphismes. Puis en milieu et derniere partie, nous nous consacrerons au developpement de l'IA. Nous allons creer un reseau de neurone utilisant le Q-learning pour apprendre. Nous fairons donc du deep semi-supervized machine learning.
